솔로 개발자 겸 여행자. Notion으로 여행 계획을 관리하며, 1인 프로젝트로 개인용 여행 대시보드를 개발 중.

- OS: Windows 11, 개발 환경은 bash (MINGW64)
- 기존 Next.js 16 + React 19 + TypeScript 5 + TailwindCSS v4 + shadcn/ui 스타터 프로젝트 보유
- 코드 스타일: 스페이스 2칸 들여쓰기, 세미콜론 없음, 함수 30줄 이하
- 커밋 메시지 한글 작성 선호
- MVP 범위를 중시하며 부가 기능은 MVP 이후로 분리하는 방식 선호
